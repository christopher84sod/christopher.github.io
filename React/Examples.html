Javascript XML

JSX betyder Javascript XML, det innehåller båda Javascript och HTML i samma fil, detta förstår inte webbläsaren så den översätts av Babel (kompileras) till Javascript. Anledningen till att man valt att blanda både Javascript och HTML är för att programmeraren lättare skall kunna förstå och kunna ändra om det behövs. Samt för att kunna färdigställa komponenter som inte är beroende av flera filer.

Router

Det finns två olika react-router verktyg som man använder mycket, BrowserRouter och HashRouter. HashRouter fungerar minst lika bra men då den är äldre och den nyare BrowserRouter används mera så använder vi den. Det själva "Router" verktyget gör så att man kan göra länkar till olika delar på sin sida. När vi använder BrowserRouter som kan man enbart ha ett "Child" element så an måste lägga allt i ett DIV.
